# EasyPostLabel

Приложение для автоматизации обработки данных о получателях и формирования почтовых этикеток.

## Описание

EasyPostLabel - это инструмент для обработки Excel-файлов с данными получателей и формирования документов для почтовой и альтернативной доставки. Приложение поддерживает различные форматы входных данных и автоматически разделяет получателей по типам доставки.

## Возможности

- Чтение и валидация данных из Excel-файлов
- Поддержка двух типов доставки:
  - Почтовая доставка (с формированием этикеток)
  - Альтернативная доставка (через пункты выдачи)
- Гибкая настройка через конфигурационные файлы
- Нормализация данных:
  - Форматирование ФИО
  - Стандартизация телефонных номеров
  - Обработка адресов доставки
- Формирование выходных документов:
  - Почтовые этикетки
  - Списки для центров выдачи
  - Сводные отчеты
- Обработка ошибок и валидация данных

## Требования

- Python 3.11+
- pandas
- openpyxl
- pytest (для тестирования)

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/yourusername/EasyPostLabel.git
cd EasyPostLabel
```

2. Создайте виртуальное окружение и активируйте его:

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Использование

1. Подготовьте входной Excel-файл с данными получателей
2. Настройте конфигурацию в файле `config.yaml`
3. Запустите обработку:

```bash
python src/main.py input.xlsx
```

## Конфигурация

Настройки приложения хранятся в `config.yaml`. Основные параметры:

- Маппинг колонок входного файла
- Настройки валидации данных
- Пути сохранения результатов
- Форматы выходных файлов
