## 🔍 Детали реализации

1. Добавлен workflow для GitHub Actions:

```yaml
name: PR Tests
on:
  pull_request:
    branches: [ develop ]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.11'
```

2. Настроены проверки:

- Запуск pytest с coverage
- Линтинг кода
- Проверка типов через mypy
